# Browsey

A simple browser built with PyQt5.

![](https://gitlab.com/PyQT/browsey/raw/master/browsey_screenshot.png)

## Installation

Get PyQT5

    pip install pyqt5

Download the repo.

## Usage

Run the script:

    python main.py

Happy browsing!
